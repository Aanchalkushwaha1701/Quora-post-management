 Built a Quora-style Post Management Web App using Express.js, REST APIs, EJS & CSS.
 
Features:
• Create new posts
• Edit existing posts
• Delete posts
• View specific post details
• Dynamic routing using RESTful APIs
• Server-side rendering with EJS
• Clean UI styling using CSS
  
Tech Stack:
Node.js | Express.js | REST API | EJS | HTML | CSS

This project helped me understand:
• RESTful routing
• CRUD operations
• Dynamic templating
• Backend request handling
• Route parameters concepts
